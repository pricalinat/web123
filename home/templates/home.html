WELCOME TO miniCTF

A Platform made in django for hosting Capture The Flag(CTF) event.

Features:

  1. Simple Interface
  2. Scoreboard
  3. Sqlite3 database
  4. Responsive Design

Challenges are available in following categories

  1. Cryptography
  2. Forensics
  3. Pwning
  4. Reverse Engineering
  5. Stegnography
  6. Web

NOTE: Pwning and Web categories are under development.
